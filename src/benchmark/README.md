- create new project
> cargo new benchmark

- Trước khi sử dụng lệnh cargo add, bạn cần cài đặt công cụ cargo-edit bằng cách chạy lệnh sau trong terminal:
> cargo install cargo-edit

- Add new package
> cargo add criterion

- Chạy chương trình bằng cách sử dụng lệnh sau:
> cargo build --release
> cargo run --release
